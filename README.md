# 🐶 Ajude meu cachorro

Este projeto é um site simples criado para apresentar meu cachorro e pedir ajuda para o tratamento de saúde dele.

## 📖 Sobre

O site conta a história do meu cachorro, mostrando:

* Fotos dele saudável 🐾
* Fotos da situação atual ⚠️
* Explicação sobre o problema de saúde
* Pedido de ajuda 🙏
* Botão para copiar a chave PIX

O objetivo é sensibilizar as pessoas e facilitar contribuições.

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (para copiar a chave PIX)

---

## 📱 Funcionalidades

* Layout responsivo (funciona bem no celular)
* Galeria de imagens organizada
* Botão de copiar chave PIX
* Design simples e emocional

---

## 📂 Estrutura do projeto

```
/projeto
│
├── index.html
├── style.css
└── /imagens
    ├── apresento1.jpeg
    ├── ...
    ├── 1.jpeg
    └── ...
```

---

## 🚀 Como usar

1. Baixe ou copie os arquivos
2. Coloque suas imagens na pasta `imagens`
3. Atualize a chave PIX no código JavaScript
4. Abra o arquivo `index.html` no navegador

---

## 💚 Como ajudar

* Compartilhando o site
* Fazendo uma contribuição via PIX
